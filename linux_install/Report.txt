1) Установила Virtual Box версию 6.1.18 для Windows, скачав установщик с сайта https://www.virtualbox.org/wiki/Downloads
2) Открыла установленный Virtual Box и нажала "Создать"
3) Назвала свою виртуальную машину VirtualMonster, выбрала, что поставлю на неё Ubuntu 64 bit.
4) Задала параметры виртуальной машины согласно рекомендациям (см. скрины). Диск сделала динамическим, чтобы не тратить на него лишнюю память.
5) Зашла на сайт, который мне показался официальным (https://ubuntu.com/download) и выбрав Ubuntu Desktop, скачала файл ubuntu-20.04.2.0-desktop-amd64.iso
